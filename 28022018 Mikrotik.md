## ¿Qué es Mikrotik?
Es un software que funciona como un Sistema Operativo para convertir un PC o una placa Mikrotik RouterBOARD en un router dedicado.

Una de las ventajas fundamental que ofrece **Mikrotik** es que funciona exactamente que un Router propietario pero a un coste significativamente inferior.

## Aplicaciones por default de _Mikrotik_ Routers
Mikrotik brinda la posibilidad de gestionar muchas aplicaciones, ya sea instalando el software en PC o entre ellas:

 1. **Enlaces Inalámbricos.**
 2. **Web Cache:** De archivos pequeños o paginas, esto no se debe realizar en un *RB* ya que por ejemplo un *RB750* no tiene la memoria suficiente para el almacenamiento.
 3. **Identificación y priorización de tráfico:** Control de tráfico, marcado de paquete.

## Configuración de Mikrotik
Antes de comenzar, debe reiniciar el enrutador. En el caso de recibir un mensaje que indique una configuración predeterminada debe eliminar la configuración.
El enrutador se debe configurar a través de _Winbox_. Winbox es la interfaz gráfica de usuario para configurar el Mikrotik Router OS.

 1. Se define el primer puerto para la conexión WAN, de forma que el enrutador se conecte a Internet a través de otro enrutador con DHCP.

En Winbox haga clic en  **IP > DHCP**  Cliente y  **Añadir cliente 		DHCP**  al puerto  **ether1**
![enter image description here](http://www.hotspotsystem.com/hotsyswp/wp-content/uploads/2013/07/dhcp-client.jpg)

 2. Se añade el servicio de zona Internet a wlan **Haga clic en IP > Zona Internet** y la casilla Zona Internet **Setup**, debe elegir **wlan1** como interfaz de zona Internet. Puede aceptar los valores predeterminados pero en el certificado elija "ninguno". Deje IP como está (10.5.50.x). Si cambia esta IP, los enlaces de INICIAR SESIÓN y CERRAR SESIÓN no funcionarán en su página de inicio.
					 ![enter image description here](http://www.hotspotsystem.com/hotsyswp/wp-content/uploads/2013/07/hotspot-wlan.jpg)

 3. El enrutador debe estar colocado en el modo puente AP.

Hacer clic en la interfaz, clic en wlan1, clic en Modo y seleccione: puente AP (ap bridge).
Se debe asegurar de que la frecuencia está establecida en 2,4 b/G.
![enter image description here](http://www.hotspotsystem.com/hotsyswp/wp-content/uploads/2013/07/radius-interface.png)

4. añadir servidor de radio como servidor de autenticación y contabilidad.

En los perfiles de zona Internet,  **(IP > Zona Internet > Perfiles)**  elija su perfil de zona Internet y haga clic en la pestaña Radio,  **marque Permitir radio**. A continuación, haga clic en la pestaña de inicio de sesión y **desmarque cookie, permitir https, http pap y chap**.
![enter image description here](http://www.hotspotsystem.com/hotsyswp/wp-content/uploads/2013/07/hotspot-radius.jpg)
![enter image description here](http://www.hotspotsystem.com/hotsyswp/wp-content/uploads/2013/07/pap.jpg)

5. Definir el servidor de radio. Hacer clic en  **Radio**  y el  **signo +**  para añadir el servidor de radio. Hacer clic en  **Servicios > Zona Internet**, introducir la dirección de **radio: Radius.hotspotsystem.com, Secret: hotsys123** Marque la casilla junto a Zona Internet.
![enter image description here](http://www.hotspotsystem.com/hotsyswp/wp-content/uploads/2013/07/radius-setup1.png)

6. Añada el servidor de radio secundario. Hacer clic en  **Radio**  y el  **signo +**. Hacer clic en  **Servicios > Zona Internet**, introduzca la dirección de **radio: Radius2.hotspotsystem.com, Secret: hotsys123** Marque la casilla junto a Zona Internet.

8. Debe sincronizar la hora del enrutador con nuestro servidor.

 11. Debe establecer las direcciones IP de la URL de inicio y cierre de sesión en el centro de control.

 13. En el último paso tendrá que añadir la comprobación horaria del estado operativo de la característica Alerta de enrutador.
